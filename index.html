Welcome to Pokemon Hatching Center!

In this game, you will be able to collect and hatch eggs in order to get all the possible Pokémon !
You should start by taking some eggs in the Shelter. Then you should go to the Interaction page to click some eggs and Pokémon !

Since the game works only with your browser (that way you'll never have to worry about the game closing), you should backup your save quite often. Or this will come in handy if you need to play in another PC. However, in order to avoid to abuse of that system, you can only export your Save every 2 hours, and import it every 8 hours !

If you have any question about the game, you should check the Help Section
