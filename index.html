Welcome to Pokemon Hatching Center!

In this game, you will be able to collect and hatch eggs in order to get all the possible Pokémon !
You should start by taking some eggs in the Shelter. Then you should go to the Interaction page to interact with some eggs and Pokémon ! Take the time to look at the help page before starting or if you have any question about the game!

Since the game works only with your browser (that way you'll never have to worry about the game closing), you should backup your save quite often. Or this will come in handy if you need to play in another PC.
